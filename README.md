🛠 Approach Notes

For each problem, I aim to:


Get a correct brute-force solution first
Optimize for time/space complexity where possible (e.g. hash maps over nested loops, two pointers over brute force, etc.)
Note key patterns/techniques used (hashing, sliding window, binary search, DP, etc.)


🚀 Why this repo exists

Practicing DSA consistently and keeping a record of solved problems, patterns learned, and edge cases handled — useful both for interview prep and for looking back at growth over time.
